| ID | Тест-кейс | Входные данные | Ожидаемый результат | Фактический результат | Статус |
|----|------------|----------------|---------------------|----------------------|---------|
| 1 | Добавление задачи | "Купить молоко" | Задача добавляется в список | Задача добавлена успешно | ✅ PASS |
| 2 | Просмотр пустого списка | Нет задач | Сообщение "Нет задач!" | Выводится корректное сообщение | ✅ PASS |
| 3 | Удаление задачи | ID существующей задачи | Задача удаляется из списка | Задача удалена успешно | ✅ PASS |
| 4 | Удаление несуществующей задачи | ID=999 | Сообщение об ошибке | Задача не найдена | ✅ PASS |
| 5 | Некорректный ввод в меню | "abc" вместо цифры | Сообщение "Неверный выбор!" | Корректное сообщение об ошибке | ✅ PASS |
| 6 | Сохранение данных | Добавить задачу и выйти | Данные сохраняются в JSON | Файл tasks.json создан | ✅ PASS |
| 7 | Загрузка данных | Перезапуск приложения | Задачи загружаются из файла | Данные загружены корректно | ✅ PASS |

# План тестирования Task Manager

## Цели тестирования

### Функциональное тестирование:
- Проверить корректность работы всех функций приложения
- Убедиться в правильности сохранения и загрузки данных
- Проверить обработку ошибочных сценариев

### Пользовательское тестирование:
- Оценить удобство интерфейса для конечных пользователей
- Выявить сложности в использовании
- Собрать предложения по улучшению

## Инструменты и среда тестирования

- **ОС**: Windows 10/11, macOS, Linux
- **Python**: версия 3.8+
- **Браузер**: для опроса - Google Forms
- **Инструменты**: встроенный unittest, ручное тестирование

## Методология

1. **Функциональное тестирование** - проверка всех функций по тест-кейсам
2. **Пользовательское тестирование** - опрос 5-10 пользователей
3. **Сбор и анализ обратной связи** - систематизация предложений
